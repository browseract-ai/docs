### API Specs

**No rate limits**

**Task limits:**

- Free users: 1 task per run
- Paid users: 20 concurrent tasks

**Session duration:** Max 7 hours per task

**Webhook support:**

- Triggered on: completion, failure, cancellation
- Must respond within 30 seconds
- Auto-retries on 5xx errors (max 3 times)

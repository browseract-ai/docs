**Q: Can I extract more than 1,000 records?**

A: Technically yes, but not recommended. Split into multiple scheduled runs for reliability.

**Q: What's the actual maximum?**

A: No hard cap, but stability issues appear \>10K records. Stick to \<1K per run.

**Q: Can I use my own LLM model?**

A: Not currently. We use an AI engine optimized specifically for web automation.

**Q: Can a credential be shared across workflows?**

A: Yes! Assign one credential to unlimited workflows from Credential Center.

**Q: Is 2FA required?**

A: No, but highly recommended for security.

**Q: Are my credentials secure?**

A: Yes. End-to-end encrypted, visible only to you. No third-party access.

**Q: What if I change my password on a website?**

A: Update it in Credential Center. Go to Edit → Update password → Save.

**Q: Can I export my workflows?**

A: Yes, as JSON files. Great for sharing and backups.

**Q: How do I cancel a running task?**

A: Dashboard → Activity Log → Click task → **"Cancel"** button
